```shell

```

## 如何使用
### 移动
- W: 前进
- A: 向左移动
- S: 后退
- D: 向右移动
- R: 开启/关闭 跑步模式
- ESC/E：释放鼠标
- Space: 跳跃
- Tab: 换模式 (飞行或走路)

右击放置方块,左击摧毁方块。

左击TNT可以让TNT爆炸。

### 方块

1. 草方块
2. 土块
3. 石头
4. 沙子
5. 木头
6. 砖块
7. 南瓜
8. 西瓜
9. TNT
